Objetivo:

Criar um novo replicador onde os dados vem de um PostgreSQL para o Unity Catalog, sem a necessidade da pré criação de tabelas ou de definição de schema. Também possui uma data de inicio caso queira copiar dados a partir de um periodo específico, também possui a possibilidade de definir o tamanho do período da query, podendo rodar um período todo, ou uma quantidade de dias ou horas em específico.

Este projeto foi construído para auxiliar momentaneamente na resolução de um problema de cargas com o replicador Airbyte.

Objetivo:

Projeto feito para realizar cargas e atualização de tabelas antes da utilização do dbt. Foi construído inicialmente para carregar de um PostgreSQL para outro PostgreSQL utilizando os notebooks do Databricks.

Com o tempo, foi atualizado para criar tabelas dentro do próprio Unity Catalog mas não foi utilizado devido ao dbt.

Modelspace são todas as funções criadas para realizar as cargas e os marts são as criações de tabela.
